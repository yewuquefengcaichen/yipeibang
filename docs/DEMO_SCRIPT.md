# 医陪帮演示脚本

## 1. 夜间出汗分层

输入：

```text
平时我同学有夜间出汗，怎么办？帮我分析一下
```

演示点：

- 意图识别：症状分层、风险追问、长期记忆
- 知识命中：夜间出汗、低血糖、感染或内分泌线索
- 写入：症状记录、风险关注记忆
- 输出：红旗信号、需要记录的字段、何时就医

## 2. 空腹血糖 8.2 + 复诊

输入：

```text
明天内分泌科复诊，空腹血糖8.2，最近夜里出汗，要带什么？
```

演示点：

- 意图识别：复诊准备、指标记录、提醒创建
- 知识命中：空腹血糖、糖尿病复诊、夜间出汗
- 写入：血糖记录、就医任务、主动提醒
- 输出：复诊材料清单、问医生的问题、家属同步提示

## 3. 血压 150/96 + 头胀

输入：

```text
最近三天血压150/96，头有点胀，帮我记录并提醒复查
```

演示点：

- 意图识别：生命体征记录、随访提醒
- 知识命中：血压、高血压、危险信号
- 写入：血压健康记录、复查提醒
- 输出：趋势记录、风险色、复查问题

## 4. 急症安全拦截

输入：

```text
我胸痛、呼吸困难、脸色发白，现在怎么办？
```

演示点：

- 路由：SAFE
- 安全层：不诊断、不开药、不等待模型
- 输出：立即 120/急诊、准备身份证医保卡、用药和过敏信息
