# 医陪帮版本修改记录

## 2026-06-17 对话体验与模型配置优化

- 对话等待阶段新增即时状态：识别意图、检索知识、连接模型、切换备用通道都会实时显示。
- 回答内容改为逐字打字动画，避免用户发送后长时间无反馈。
- 优化 Markdown 渲染：标题、编号列表、项目符号、重点文字改成更清晰的产品化样式。
- 修复普通症状类问题的意图识别，不再把“恶心、出汗、头晕”等问题默认归到复诊准备。
- 修复问候语场景，`hi/你好` 不再被历史对话带偏。
- 底部模型状态只显示供应商名称，不展示具体模型名。
- 模型调用失败时自动展示切换过程，优先保证仍可继续真实 API 对话。
- 修复侧栏收起状态和移动端菜单展示。
- 增加回答区底部留白，减少固定输入框遮挡长回答。

## 2026-06-17 病例库、图片资源与功能页交互增强

- 新增 `assets/medical/` 本地医疗健康图片资源，病例库、知识库、健康记录、提醒中心均使用真实图片展示。
- 知识库重构为六个入口：病例库、症状、疾病、体检指标、用药提醒、生活方式。
- 新增 12 个高质量病例库样例，每个病例绑定图片、患者场景、标签和可直接提问的演示问题。
- 新增 60+ 条分层医疗知识，覆盖症状观察、慢病复诊、体检指标、用药沟通和生活方式管理。
- “我的档案”支持新增和编辑长期记忆。
- “健康记录”支持新增记录，并可一键让医陪帮分析。
- “提醒中心”支持创建提醒、标记完成、生成执行清单。
- 侧栏视觉改为更接近参考图的暖白产品风格，增加产品名、副标题和新建对话按钮。
